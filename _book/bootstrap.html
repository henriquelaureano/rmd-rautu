Bootstrap

Bootstrap strategies compose a class of non-parametric Monte Carlo methods
that estimate the distribution of a population by means of resampling.

Resampling methods treat samples as a finite population, from which samples are taken to estimate features and make inferences about this population.

Example 1 Suppose we observe the following data:

\(x = {2, 2, 1, 1, 5, 4, 4, 3, 1, 2}\).

Resampling from \(x\), we select \(\{1, 2, 3, 4, 5\}\) with probabilities \(\{0.3, 0.3, 0.1, 0.2, 0.1\}\). The \(F_{X^{*}}(x)\) distribution of a sample taken at random is exactly the function \(F_{n}\):

\[ F_{X^{*}}(x) = F_{n}(x) = \begin{cases} 0.0 & x < 1; \\ 0.3 & 1 \leq x < 2; \\ 0.6 & 2 \leq x < 3; \\ 0.7 & 3 \leq x < 4; \\ 0.9 & 4 \leq x < 5; \\ 1.0 & x \geq 5. \end{cases} \]

Bootstrap estimation of standard error

Example 2 (Correlation) 
lattice::xyplot(dist ~ speed, cars, pch = 16, ylab = list(rot = 0))

Figure 1: A scatterplot of the speed and dist variables of the cars dataframe. 

set.seed(1)
B <- 10e3
boot.corr <- vector('numeric', B)
for (b in 1:B){
  ind <- sample(nrow(cars), replace = TRUE)
  boot.corr[b] <- with(cars[ind, ], cor(dist, speed))
}
(theta.star <- mean(boot.corr))
[1] 0.8062458
(se.theta.star <- sd(boot.corr))
[1] 0.04782856
